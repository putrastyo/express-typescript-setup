# Sakuin

Dokumentasi ini berisi panduan setup teknis untuk pengembangan backend menggunakan Node.js, Express, dan TypeScript.

## Tech Stack

- **Runtime:** Node.js (v20+)
- **Language:** TypeScript
- **Framework:** Express
- **Database:** MongoDB (via Mongoose)
- **Testing:** Vitest & Supertest
- **DevOps:** Docker, Docker Compose, GitHub Actions
